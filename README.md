
# Tech Talks

In diesem Repository finden Sie die benötigten Dateien, um in den Tech Talks praktisch mitarbeiten zu können. Die Teilnahme ist selbstverständlich aber auch ohne das Beispielprojekt möglich.

## Wo finde ich die Stände voriger Tech Talks?

Das Repository hat mehrere Branches. Die aktuellste Version finden Sie immer im `master` Branch, Stände von vorigen Tech Talks finden Sie auf GitHub oben links unter dem Dropdown "Branches". Wenn Sie das Projekt in Ihrer lokalen Entwicklungsumgebung geklont haben, natürlich auch in Ihrer favorisierten IDE.

## Wie kann ich das Projekt klonen?

Wenn Sie selber mit den Dateien arbeiten möchten, können Sie sich das Repository lokal clonen. Dafür benötigen Sie eine installierte Version von [GIT](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git), dem Versionsverwaltungsprotokoll, auf dem GitHub oder auch GitLab aufbauen. Nach der Installation führen Sie in der Kommandozeile einfach `git clone https://github.com/nordlb/techtalks.git` in dem gewünschten Zielordner aus. Sie können nun in dem Projekt arbeiten, ihre Änderungen feststellen und weitere Funktionen des GIT-Protokolls nutzen.

### Kann ich auch ohne GIT lokal an den Dateien arbeiten?

Die Verwendung von GIT bietet viele Vorteile: Änderungen für zukünftige Tech Talks können im Repository automatisch heruntergeladen ("gepullt") werden. Wer sich GIT aber nicht extra installieren möchte, kann die Dateien auch einfach als ZIP herunterladen. Auch die von Ihnen gemachten Veränderungen können dann aber nicht sichtbar gemacht werden.

## Wie kann ich in dem Projekt arbeiten?

Das lässt sich am besten mit einer IDE (Integrated Development Environment), einer integrierten Entwicklungsumgebung. Als Entwickler in der NORD/LB benutzen Sie vermutlich [IntelliJ IDEA](https://www.jetbrains.com/idea/). Wenn Sie bisher keine integrierte Entwicklungsumgebung nutzen, empfiehlt sich [Visual Studio Code](https://www.jetbrains.com/idea/), eine kostenlose und einfach gehaltene Lösung. Neben VS Code gibt es aber auch einige andere Alternativen, die alle Funktionen bieten, die Sie in den Tech Talks benötigen werden.
